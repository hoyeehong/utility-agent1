HOW IT WORKS
════════════════════════════════════════════════════════════════════════════

1. FastAPI starts → Initializes TariffRateManager
2. TariffRateManager connects to Supabase (or uses fallback)
3. UtilityCalculator fetches dynamic rates from manager
4. Bill calculations use current rates (automatic updates)
5. Admin CLI allows manual rate updates when SP/PUB announce changes

RATE UPDATE PROCESS (Quarterly)
════════════════════════════════════════════════════════════════════════════

When SP Energy or PUB announces new rates:

# Step 1: Update rate
python cli_admin.py update-rate electricity 0.2750

# Step 2: Verify
python cli_admin.py get-rates

# Step 3: Done ✅
- All new bills automatically use updated rate
- Previous bills unchanged
- Full audit trail maintained
